# 关于 KDoc 风格文档主题

KDoc 是一款极简、现代、Ruby 官网色调的 Jekyll 文档主题，适用于企业/开源项目文档、API 文档、团队知识库等场景。

## 主题特性
- 顶部导航栏，支持多入口
- 左侧分组菜单，支持多模块文档
- 右侧目录（TOC），自动生成
- 代码高亮，支持多种语言
- 响应式设计，适配桌面和移动端
- 极简 Ruby 红色调，现代卡片风格

## 适用场景
- 项目/开源文档
- API/SDK 文档
- 团队知识库
- DevOps/运维平台文档
